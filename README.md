# Git Mastery: DevOps C App 🚀

A modular C-based utility program developed to demonstrate professional Git workflows, branching strategies, and manual conflict resolution.

## 🎯 Project Objectives
- **Version Control:** Comprehensive tracking of software evolution.
- **Branch Management:** Parallel development using `feature-calc` and `feature-menu` branches.
- **Conflict Resolution:** Identified and resolved manual merge conflicts during feature integration.
- **DevOps Integration:** Connecting local development environments to GitHub for CI/CD readiness.

## 🛠️ Technical Features
- **Greeting Module:** Captures user input for personalized interaction.
- **Calculator Engine:** Implements addition logic via a modular function.
- **Menu Interface:** A structured UI for navigating application features.



## 🚀 Installation & Usage
To compile and execute this project on Ubuntu:

```bash
# Clone the repository
git clone [https://github.com/Yasmeen-T/devops-c-app.git](https://github.com/Yasmeen-T/devops-c-app.git)
cd devops-c-app

# Compile using GCC
gcc app.c -o devops-app

# Run the binary
./devops-app
